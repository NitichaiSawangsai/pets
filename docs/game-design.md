# Game Design

## Required rules implemented

1. Pet can become hungry, lonely, sick, and can die.
2. Pet food is finite and must be bought with coins.
3. Coins are earned by caring for the pet.
4. Focus mode makes the pet less distracting while work is active.
5. The pet can be moved around the screen on macOS.
6. Local-first state is mirrored to iCloud Drive when available.

## Optional rules worth adding later

1. Personality traits: shy, playful, sleepy, dramatic.
2. Weather moods using local-only generated weather states.
3. Tiny quests: clean room, teach trick, bedtime streak.
4. Evolution stages based on care quality, not just time.
5. Gentle work mode: pet sleeps and only asks once per interval.
6. One daily rare snack to create a habit without spam.
